Welcome to the FDM Trainer Skills and Availability

Scheduling Team members

The scheduling team members will be responsible for creating courses and creating modules that will be added to the courses. They will also be responsible for booking a trainer to these created courses.

Trainers

The trainers will be able to login and access their week schedule which will contain all of the booked times that the team scheduler has made.

Trainees

The trainees will be the ones who will be attending the booked modules either online or in person.
